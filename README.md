# Spring Boot GitHub Actions

A simple Spring Boot application demonstrating CI/CD automation using GitHub Actions. This project showcases how to set up automated build, test, and deployment pipelines for a Java Spring Boot application.

## 📋 Project Overview

- **Framework**: Spring Boot 4.0.3
- **Language**: Java 21
- **Build Tool**: Maven
- **Container**: Docker
- **CI/CD**: GitHub Actions

## 🚀 Getting Started

### Prerequisites

- **Java 21** or higher
- **Maven 3.6+**
- **Docker** (for containerization)
- **Git** (for version control)
